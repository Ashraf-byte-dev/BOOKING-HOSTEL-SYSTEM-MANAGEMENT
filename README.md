# Booking Hostel System Management

A comprehensive web-based hostel booking and management system built with **HTML, Java, and PHP**.

## Features
- User registration and authentication
- Room booking management
- Payment processing
- Admin dashboard
- Guest reviews and ratings

## Project Structure
```
BOOKING-HOSTEL-SYSTEM-MANAGEMENT/
├── frontend/          # HTML & CSS files
├── backend/           # PHP backend
├── java/              # Java services (optional business logic)
├── database/          # SQL scripts
├── config/            # Configuration files
└── docs/              # Documentation
```

## Tech Stack
- **Frontend:** HTML5, CSS3
- **Backend:** PHP 7.4+
- **Database:** MySQL/MariaDB
- **Java:** For additional services (if needed)
- **Server:** Apache/Nginx

## Installation & Setup

### Prerequisites
- XAMPP, WAMP, or LAMP stack installed
- PHP 7.4 or higher
- MySQL 5.7+
- Java JDK 11+ (optional)

### Quick Start
1. Clone the repository
2. Set up the database (see `database/setup.sql`)
3. Configure database credentials in `config/db.php`
4. Start your local server
5. Access via `http://localhost/BOOKING-HOSTEL-SYSTEM-MANAGEMENT`
